# React Basics

React + TypeScript + Vite project with React Router and Bootstrap styling.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open the app URL shown in terminal (usually `http://localhost:5173`).

## Bootstrap setup

- Bootstrap is installed via npm package `bootstrap`.
- Global Bootstrap CSS is loaded in `src/main.tsx`:

```ts
import 'bootstrap/dist/css/bootstrap.min.css'
```
